# 🛒 Shopping List App

아이템 추가, 삭제, 체크 기능이 있는 심플한 쇼핑 리스트 웹 앱입니다.

## 기능

- ✅ 아이템 추가 (버튼 클릭 또는 Enter 키)
- ✅ 아이템 체크/체크 해제
- ✅ 아이템 개별 삭제
- ✅ 완료 항목 일괄 삭제
- ✅ localStorage로 데이터 영구 저장
- ✅ 총 개수 / 완료 개수 실시간 표시

## 실행 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 실행됩니다.

## 테스트 실행

Playwright를 사용한 자동 테스트가 포함되어 있습니다.

```bash
npm install
node shopping-test.mjs
```

### 테스트 항목 (총 21개)

| 테스트 | 항목 수 |
|--------|--------|
| 초기 상태 확인 | 3 |
| 아이템 추가 (버튼) | 3 |
| 아이템 추가 (Enter) | 2 |
| 빈 입력값 방지 | 1 |
| 체크 기능 | 3 |
| 체크 해제 | 2 |
| 개별 삭제 | 2 |
| 일괄 삭제 | 3 |
| localStorage 유지 | 2 |

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라)
- Playwright (자동 테스트)
